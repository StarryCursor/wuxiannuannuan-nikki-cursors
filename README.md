## 项目内容 / Project Overview
这是一套以《无限暖暖》为主题制作的鼠标指针皮肤。以在《无限暖暖》中的暖暖的形象和元素为原型，由StarryCursor设计并制作，包含13种鼠标状态。

支持：
- Windows `.cur` 鼠标指针方案
- macOS Mousecape `.cape` 
- 多尺寸高清指针


## 安装 / Installation
### MAC：
1. 下载并安装 [Mousecape](https://github.com/alexzielenski/Mousecape/releases)
2. 打开 Mousecape，菜单栏 `Mousecape` → `Install Helper Tool`
3. 下载本仓库的 [无限暖暖03_Mac.cape](https://github.com/StarryCursor/wuxiannuannuan-nikki-cursors/blob/main/%E6%97%A0%E9%99%90%E6%9A%96%E6%9A%9603_Mac.cape) 文件
4. 双击 .cape 文件导入到 Mousecape
5. 右键导入的 cape → `Apply`

【Mac可用的 26 个光标】
- arrow          正常箭头
- ibeam          文本选择
- pointing       链接选择
- closed         拖拽结束
- open           拖拽开始
- forbidden      不可用
- help           帮助
- wait           忙碌                     
- window e       向右调整 → 
- window w       向左调整 ←
- window n       向上调整 ↑
- window s       向下调整 ↓
- window e-w     水平调整 ↔
- window n-s     垂直调整 ↕
- window ne      右上调整 ↗
- window nw      左上调整 ↖
- window se      右下调整 ↘
- window sw      左下调整 ↙
- window ne-sw   对角调整 ↗↙
- window nw-se   对角调整 ↖↘
- 每个内嵌 4 种分辨率 (48/96/240/480)，自适应 100%~300% 显示缩放。
   
### Windows:
下载本仓库的zip [无限暖暖-02-Windows-v6.zip](https://github.com/StarryCursor/wuxiannuannuan-nikki-cursors/blob/main/%E6%97%A0%E9%99%90%E6%9A%96%E6%9A%9603_Windows_CUR%E5%85%89%E6%A0%87%E5%8C%85.zip)，解压全部文件。

【快速安装方法】
1. 打开 `Recommended_Windows_Scheme` 文件夹。
2. 右键单击 install.inf (Windows 11 用户：先点"显示更多选项"或按 Shift+F10)
3. 在菜单里点"安装"
4. 打开 Windows 系统设置，进入“蓝牙和设备 → 鼠标 → 其他鼠标设置”。
5. 打开“指针”选项卡。
6. 选择对应的无限暖暖鼠标指针方案，点击“应用”。
   
【若无法快速安装】
1. 下载后，打开系统设置-辅助功能-鼠标指针与触控-自定义指针图像/自定义指针样式。
2. 针对每一个状态的光标，点击“浏览”，打开下载的文件夹中 Cursors 文件，选择其对应状态的光标，进行一一替换。
  
【Windows可用的 12 个光标】
- 01_Normal_Select.cur                     正常箭头
- 02_Text_Select.cur                       文本选择
- 03_Busy.cur                              忙碌/工作中
- 04_Link_Select.cur                       链接选择
- 05_Unavailable.cur                       不可用
- 06_Help_Select.cur                       帮助选择           
- 07_Move_SizeAll.cur                      移动                     
- 08_Horizontal_SizeWE.cur                 水平调整 ↔
- 09_Vertical_SizeNS.cur                   垂直调整 ↕
- 10_Diagonal2_SizeNESW.cur                对角调整 2 (NE-SW) ↗↙
- 11_Diagonal1_SizeNWSE.cur                对角调整 1 (NW-SE) ↖↘
- 12_Background_Working_AppStarting.cur    后台运行
- 每个 .cur 内嵌 6 种分辨率 (32/48/64/96/128/256)，自适应 100%~300% 显示缩放。
